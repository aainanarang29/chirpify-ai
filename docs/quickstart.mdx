# Quickstart

Get Chirpify AI running on your local machine.

## Prerequisites

- Node.js 18+
- npm or yarn
- ElevenLabs API key
- Dodo Payments API key (for payments)

## Installation

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/aainanarang29/chirpify-ai.git
    cd chirpify-ai
    ```
  </Step>

  <Step title="Install dependencies">
    ```bash
    npm install
    ```
  </Step>

  <Step title="Set up environment variables">
    Create a `.env` file:
    ```bash
    ELEVENLABS_API_KEY=your_elevenlabs_key
    DODO_PAYMENTS_API_KEY=your_dodo_key
    DODO_ENV=test
    ```
  </Step>

  <Step title="Start the server">
    ```bash
    npm start
    ```
  </Step>

  <Step title="Open in browser">
    Visit [http://localhost:3000](http://localhost:3000)
  </Step>
</Steps>

## Try It Out

1. Type some text in the textarea
2. Select a voice
3. Click **Speak** to generate audio
4. Your 500 free characters will be deducted

## Deploy to Vercel

<Steps>
  <Step title="Push to GitHub">
    ```bash
    git push origin main
    ```
  </Step>

  <Step title="Import to Vercel">
    Go to [vercel.com/new](https://vercel.com/new) and import your repo
  </Step>

  <Step title="Add environment variables">
    Add your API keys in Vercel's dashboard under Settings â†’ Environment Variables
  </Step>

  <Step title="Deploy">
    Vercel will automatically deploy on every push
  </Step>
</Steps>
