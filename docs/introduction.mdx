# Welcome to Chirpify AI

Chirpify AI is a modern text-to-speech web application that charges users per character using Dodo Payments.

## Pricing

| Pack | Characters | Price | Per 1K Chars |
|------|------------|-------|--------------|
| Starter | 10,000 | $5 | $0.50 |
| Pro | 50,000 | $10 | $0.20 |
| Power | 200,000 | $25 | $0.125 |

## Tech Stack

- **Frontend**: Vanilla HTML/CSS/JS
- **Backend**: Node.js + Express (Vercel serverless)
- **TTS API**: ElevenLabs
- **Payments**: Dodo Payments
